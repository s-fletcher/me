CONTEXT

The purpose of this project was to take the Raspberry Pi 3 and create a functional security camera that is activated by motion. I used an Infrared PIR Motion Sensor to detect motion and snap a picture using a 5MP Camera Module attached to the Pi. The picture is saved to the Pi and emailed to a desired gmail account with a time stamp. I created this as a final project during my first semester in college. It went on to win first place out of ten other projects in my freshman class. This was my first time working with Python and a Raspberry Pi.
